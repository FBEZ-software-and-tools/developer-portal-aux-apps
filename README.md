# developer-portal-aux-apps

Repo containing some apps to speed up article creation in the developer portal


## Get started

1. Fork the repo on github (or gitlab)
2. Move inside the repo folder: `cd repo`
3. Clone the forked repo (inside the `repo` folder): 
    ```bash
    git clone --recursive --shallow-submodules <your_username/your_fork_repo_name>
    ```
4. Run docker compose: `docker compose up -d`
